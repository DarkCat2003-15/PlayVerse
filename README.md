# 🎮 PlayVerse

> Proyecto Final desarrollado durante la **Mentoría Bootcamp Full Stack Developer**.

## 📖 Descripción

PlayVerse es una plataforma web inspirada en Steam, diseñada para la gestión y distribución de videojuegos digitales. El proyecto fue desarrollado aplicando una arquitectura Full Stack, integrando un frontend moderno, una API REST en el backend y una base de datos relacional.

El objetivo principal fue poner en práctica los conocimientos adquiridos durante el bootcamp, implementando buenas prácticas de desarrollo, consumo de APIs, persistencia de datos y organización del código.

## 🛠️ Instalación

### 1. Clonar el repositorio

```bash
git clone https://github.com/DarkCat2003-15/PlayVerse.git
```

---

### 2. Backend

Ingresar al proyecto

```bash
cd BackEnd/Steam-Clone
```

Abrir con Visual Studios.

Configurar la conexión a la base de datos.

Ejecutar:

```
SteamCloneApplication
```

---

### 3. Base de Datos

Ejecutar los scripts ubicados en:

```
Base de datos/
```

---

### 4. Frontend

Abrir la carpeta desde Visual Code

```
FrontEnd/SteamClone
```

Ejecutar el proyecto mediante Live Server o el servidor de preferencia.
